Check out these example apps to gain real-world knowledge on how IHP apps are built.

## Official Examples

### IHP Chat: Basic slack-like chat application

A fully functional chat application demonstrating real-time features, authentication, and database operations.

- [Source Code on GitHub](https://github.com/digitallyinduced/ihp-chat-example-app)
- [Develop this project on GitPod](https://gitpod.io/#https://github.com/digitallyinduced/ihp-chat-example-app)

<img src="https://github.com/digitallyinduced/ihp-chat-example-app/raw/master/static/screenshot.png" alt="IHP Chat App" />

### IHP Blog: A simple markdown-based blog application

This project contains the blog from the "Your First Project" guide, showing the basics of CRUD operations, routing, and views.

- [Source Code on GitHub](https://github.com/digitallyinduced/ihp-blog-example-app)
